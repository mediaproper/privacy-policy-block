Displays a privacy policy link if the privacy policy page is set.

== Description ==

This block offers a straightforward solution for WordPress site owners to display a privacy policy link. Once the privacy policy page is set up in your WordPress settings, this plugin automatically generates a link to that page, which can be easily inserted anywhere on your site using the provided block. This ensures that your users can always find and access your privacy policy easily.

== Installation ==

1. Download the "Privacy Policy Block" plugin from the WordPress plugin repository or directly through your WordPress dashboard.
2. Navigate to the WordPress admin area and go to Plugins > Add New. Click the 'Upload Plugin' button at the top of the page.
3. Select the plugin file from your computer and click 'Install Now'.
4. After the plugin has been installed, click on the 'Activate Plugin' link to activate it.
5. Go to the settings page of the plugin (if applicable) to configure any options according to your needs.
6. Use the "Privacy Policy Block" by adding it to any post or page through the Gutenberg editor. Simply search for the "Privacy Policy Block" in the block inserter to add it to your content.

For more detailed instructions and usage tips, please visit the official documentation page of the plugin.
